# Experiment-1-HELLO-WORD
Aim

To create a simple Android application that displays “Hello World” on the screen.

Algorithm

1. Start Android Studio.


2. Create a new Android project.


3. Select Empty Activity template.


4. Open activity_main.xml file.


5. Add a TextView control in the layout.


6. Set the text property of TextView as “Hello World”.


7. Open MainActivity.java file.


8. Set the layout using setContentView().


9. Run the application on Android Emulator or Mobile device.


10. The message Hello World is displayed on the screen.
